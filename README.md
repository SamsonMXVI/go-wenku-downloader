# 轻小说文库下载器

## !!! 本项目参考 https://github.com/Messiahhh/wenku8-downloader 开发 !!!

![](assets/1.png)

## 介绍

本工具可用来下载[轻小说文库](https://www.wenku8.net/index.php)的小说，支持以下功能

- 支持生成*epub*格式的电子书
- 支持下载轻小说文库站点已下架的小说，如《无职转生》
